启动Tomcat

```shell
sudo sh /Users/hzx/devTools/apache-tomcat-9.0.58/bin/startup.sh
sudo sh startup.sh
```

关闭Tomcat

```shell
sudo sh /Users/hzx/devTools/apache-tomcat-9.0.58/bin/shutdown.sh
sudo sh shutdown.sh
```
